Implement and simulate a RISC-V processor in System Verilog. 

All Phases must be completed correctly, and the score for the
phase will be a function of the time it is submitted correctly, much like an Engineering project in industry. The Project
must be completed independently by each student.
• Phase 1: Tooling (5%)
• Phase 2: Decoding processor instructions (Verilog practice) (10%)
• Phase 3: Framework for a complete 5-stage CPU. (10%)
• Phase 4: Full implementation of two instructions (add/addi) (20%)
• Phase 5: Pipeline forwarding (20%)
• Phase 6: ALU, Upper Immediate, and Branch instructions. (20%)
• Phase 7: Memory Control (15%)
